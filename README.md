# snakes-and-ladders-pygame
A Snakes and Ladders Game made with Python 3.7 using module pygame.

## steps to Play the Game
  - ##### Install Python

    - https://www.python.org/downloads/

  - ##### Install PYgame
    - ```pip install pygame```

  - ##### Run Python File
    - ``` python "Snakes-Ladders-Final.py" ```
